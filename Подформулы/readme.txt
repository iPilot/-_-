Файл data.txt содержит данные о наборах разрешенных символов:
1-а¤ строка - алфавит символов используемых в именах переменных(a..zA..Z по-умолчанию)
2-а¤ строка - бинарные операции, разделенные пробелами на группы по приоритетам в порядке его уменьшения
3-¤ строка - унарные операции(обладают наивысшим приоритетом) (~! по-умолчанию)
4-ая строка - символы группировки идут парами начальный-конечный (()[]{} по-умолчанию)

Внутри строки формулы допустимы пробелы. \
